# Gym Streak Tracker

This is a Python-based console application that tracks gym attendance streaks.

## Features
- Accepts daily gym attendance as input
- Calculates current gym streak
- Calculates longest gym streak

## Technologies Used
- Python
- Lists
- Loops
- Conditional statements

## Learning Outcome
- Improved problem-solving using loops
- Learned streak calculation logic
- Practiced basic Python data structures
